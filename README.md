# Custom AVL Map (C++)

This is a custom implementation of an ordered map based on an AVL tree.

## Features
- AVL self-balancing tree
- Insert / Erase operations
- In-order iterators
- operator[]
- Move semantics (Rule of 5)
- Deep copy (clone)
- Parent pointers for iterator traversal

## Complexity
- Insert: O(log n)
- Erase: O(log n)
- Find: O(log n)

## Notes
This project is for educational purposes and mimics std::map behavior.
